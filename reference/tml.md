# Target make and load

Convenience function to make and load an object when working
interactively with a targets workflow. Calls \`targets::tar_make(x)\`
then \`targets::tar_load(x)\`

## Usage

``` r
tml(x)
```

## Arguments

- x:

  Object name to make and load

## Value

Nothing

## Author

Gerry Ryan

## Examples

``` r
if (FALSE) { # \dontrun{
tml(x)
} # }
```
